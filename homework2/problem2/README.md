# Problem2

## 时间复杂度

### Inputword()

    Inputword()函数是将输入进来的单词一个字母一个字母存进题目中所要求的单链表，个人觉得题目本意并不是倒着存，所以这里采用了正着存的方法，时间复杂度为O(1)。

### printSameSuffix()

    printSameSuffix()函数是本题目的核心处理函数，采用的方法只将链表遍历列一次，所以时间复杂度为O(n)。

### print() and clear()

    这两个函数都是只遍历的一次。时间复杂度为O(n)。

### 综上所述，本代码的时间复杂度为O(n)

## 空间复杂度

### 本题目要求使用单链表，所以空间复杂度很简单就是O(n)
